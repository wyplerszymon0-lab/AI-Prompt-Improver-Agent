# AI-Prompt-Improver-Agent
A modular AI agent that uses OpenAI API to transform simple ideas into professional prompts.
## Files
- `main.py`: Main entry point.
- `agent.py`: Prompt Engineering logic.
- `memory.py`: Saves history to `history.json`.

## How to run
1. Install requirements: `pip install openai`
2. Add API Key in `main.py`.
3. Run: `python main.py`
